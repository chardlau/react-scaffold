# React scaffold

A react scaffold for beginner.

## How to use
```
git clone git@github.com:chardlau/react-scaffold.git <directory-to-save>
cd <directory-to-save>
npm install --registry=https://registry.npm.taobao.org
npm start
```

## Build
```
npm run build
```

## Preview build result
```
npx serve dist
```
